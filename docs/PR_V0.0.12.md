# PR: V0.0.12 图生图、扩图、调用分级与账号昵称

## 背景

V0.0.11 已补齐生成图和编辑图的水印确定性叠加。本次 V0.0.12 继续解决实际制图流程中的连续迭代问题：用户做完第一版后，需要基于上一版继续整图修改、局部修改或扩图，而不是每次重新文字生图。

## 本次改动

- AI 工作台新增四种制作模式：文字生图、整图修改、局部修改、扩图。
- 整图修改可选择当前图或历史图作为底图，进入 `IMAGE_EDIT` 改图链路。
- 局部修改保持现有蒙版链路，用户选择模式后进入局部重画。
- 扩图支持向左、向右、向上、向下、四周扩展，并记录扩展比例。
- 素材调用分为三类：
  - 只参考：使用模板库参考图，只影响风格、色系、构图。
  - 绝对调用：素材内容不允许 AI 修改，但显示尺寸和位置可由用户手动缩放、移动。
  - 适配调用：素材必须出现，可锁比例缩放，并允许按品牌色系调整。
- 素材库文件支持拖入画布，成为可移动、可缩放的画布图片元素。
- 水印配置支持直接拖拽位置和拖右下角调整显示尺寸，不再只依赖 X/Y 滑杆。
- 工作台未提交内容支持本地草稿恢复，离开页面后再回来会恢复文字、模式、选项、参考图和水印配置。
- 账号设置新增昵称修改；首页「你好，XXX」和侧栏用户信息读取服务端昵称。
- 工程版本同步为 `0.0.12`。
